# Site Portifólio Kelvyn Emanoel
Bem vindo ao meu site portifólio! O projeto tende a documentar todo o processo de criação do site, usando ferramentas como HTML e CSS, tendo atenção a aplicação de uma boa User Interface e User Experience. A ideia também é aplicar a responsividade ao site, fazendo com que alcançe mais pessoas e todos tenham uma boa experiência ao navegar por ele.
